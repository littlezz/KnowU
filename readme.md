KnowU
================
这个参加是某公司~~创业大赛~~编程马拉松比赛的作品,花了比较多的心血,也付出了很多的时间,最后两天在现场的时候通宵写程序,30多个小时内只睡了半个多小时.  

主要的功能是根据tag推文章.文章来自后端通过rss来抓取.    
有收藏夹功能,点赞功能.  

不知道以后会不会维护这个项目  

开发
-------------
后端使用python3 和django开发.  
前端使用html5,jquery javascript, css没有使用框架,纯手写.
